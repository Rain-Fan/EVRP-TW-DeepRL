# Day 1: Project Understanding
construct a model to solve the problem of VRP, VRPTW, and EVRP-TW by using neural network and reinforcement learning algorithm which is able to learn from experience automatically

## What is VRP?
a plan that aim to minimize the time of the vehicle to visit a place

## What is VRPTW?
a plan that aim to minimize the time of the vehicle to visit a place which has a time window

## What is EVRP-TW?
a plan that aim to minimize the time of the electric vehicle to visit a place which has a time window and a electricity demand

## Why Deep RL may be useful?
